¿Cómo Reportar una moscota perdida?

Para reportar una mascota que se haya extraviado se debe ingresar a la sección Mascostas Perdidas donde se encontrará un enlace a la página para reportar mascotas perdidas, allí aparecerá el formulario para el registro de mascotas perdidas, diligencia todos los datos y selecciona la opción confirmar para enviar el formulario.

¿Cómo Reportar una mascota encontrada?

Para reportar una mascota encontrada se debe ingresar a la sección Mascostas Encontradas donde se encontrara un enlace a la página para reportar mascotas encontradas, alli aparecerá el formulario para el registro de mascotas encontradas, diligencia todos los datos y selecciona la opción confirmar para enviar el formulario.

¿Cómo Reportar una mascota en Adopción?

Para reportar una mascota que este en adopción se debe ingresar a la seccion Mascostas en Adopción donde se encontrara un enlace a la página para reportar mascotas en adopción, alli aparecerá el formulario para el registro de mascotas en adopcion, diligencia todos los datos y selecciona la opción confirmar para enviar el formulario.

¿Cómo comentar sobre una mascota?

Para realizar un comentario de una mascota se debe ingresar a cualquiera de las secciones y seleccionar la mascota a la que se le quiere realizar el comentario, dando click en la foto o en el nombre de la mascota, allí apareceran todos los datos de la mascota, y el campo para realizar un comentario, introduce el comentario y pulsa en boton comentar.
